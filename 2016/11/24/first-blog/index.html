如何采用Hexo + github搭建自己的博客站点

发表于 2016-11-24 |

　　现在很多人工作久了就想写博客了，一方面可以帮助自己来记录整理知识点，另一方面还可以很好的帮助到别人了解到这方面的知识，所以搭建一个自己的小窝很有必要的，话不多说，现在开始．ps:这篇帖子暂时只是针对windows的

一　需要安装的软解

　　1.　node.js （用来生成静态页面）
　　2.　git （把本地内容提交到github上去）
　　3.　github仓库 （用来做博客的远程创库、域名、服务器之类的）

二　以上软件安装

　　1.　node.js安装:
　　　　根据自己的系统类型在官网下载就行http://nodejs.cn/download/ 这个是node.js中文网下载，访问速度比较快，但是版本好像比官网的低一些，如果你的电脑王文国外的网站比较快，可以在http://nodejs.org/en/download/上面下载高版本的node.js
　　　　安装的方法很简单，选择好安装路径后，一路下一步就好，软件里面已经集成了npm.

　　2.　git安装:
　　　　同样的是根据自己的系统类型去官网下载就好https://git-scm.com/downloads，安装也是很简单的，网上很多教程

三　安装hexo

　　1.　执行以下命令安装hexo
　　　　$ sudo npm install -g hexo

　　　　如果安装时出现了如下错误：
　　　　　　

　　　　可能是被墙了，换成国内的镜像试试
　　　　$ npm config set registry="http://registry.cnpmjs.org"　　

　　　　然后再执行试试
　　　　$ sudo npm install -g hexo

　　2.　初始化hexo，在你本地随便哪里创建一个文件夹，如：MyBlog，然后cd 到MyBlog文件夹下，执行
　　　　$ hexo init　　

　　　　这个时候对hexo初始化就完成了　　

　　3.　生成静态html文件，执行
　　　　$ hexo generate （hexo g 也可以）　　

　　4.　启动服务
　　　　$ hexo server　　

　　　　浏览器输入http://localhost:4000，这个时候不出意外就应该看到了页面.

　　5.　配置github仓库

　　　　注意：repository: 填写的就是上面保存的github仓库的地址，另外，每个 : 后面都有一个空格.

　　6.　执行如下命令才能使用git部署
　　　 $ npm install hexo-deployer-git --save

　　7.　执行如下命令发布就好了
　　　 $ hexo deploy　　

　　　在浏览器输入：http://learnerli.github.io/就能访问了，注意，要将链接中的learnerli换成你自己的github用户名　　

　　8.　以后每次部署，就只要执行如下命令基本上就可以了
　　　　$ hexo clean
　　　　$ hexo generate
　　　　$ hexo deploy

四　主题

　　1.在这里有大量的主题https://github.com/hexojs/hexo/wiki/Themes，大家可以选择自己喜欢的主题，进行安装，安装完了，别忘记发布.
